# Документация проекта «todo-highlighter»

**Автор:** Мясникова Софья Кирилловна (М3106)  
**Проект:** Плагин «todo-highlighter» 
**Функциональность:** автоматическая подсветка комментариев TODO / FIXME / HACK

---

# 1. Назначение проекта

Расширение предназначено для улучшения читаемости исходного кода.

Типы поддерживаемых комментариев:

| Тип    | Назначение                     | Пример                 |
|--------|--------------------------------|-------------------------|
| TODO   | задача на будущее              | `// TODO: исправить`   |
| FIXME  | известная ошибка               | `// FIXME: null check` |
| HACK   | временное решение              | `// HACK: костыль`     |

---

# 2. Архитектура и принцип работы

Расширение использует механизм **TextEditorDecorationType**
встроенный в VS Code API.

### Основные компоненты:

- **extension.js** — логика расширения, поиск комментариев и применение стилей.
- **package.json** — команды, активация.
- **README.md** — описание для пользователей.
- **DOCUMENTATION.md** — техническое описание.

### Схема работы:

```text
Пользователь открывает файл
          │
          ▼
VS Code активирует расширение
          │
          ▼
activate()
  ├─ создаёт три стили подсветки
