# Документация проекта «todo-highlighter»

**Автор:** Мясникова Софья Кирилловна, группа М3106  
**Проект:** Расширение Visual Studio Code  
**Назначение:** подсветка специальных комментариев TODO / FIXME / HACK в исходном коде

---

## 1. Назначение и область применения

Расширение `todo-highlighter` предназначено для упрощения навигации по исходному коду за счёт автоматической подсветки специальных комментариев, которые разработчики используют как пометки:

| Тип    | Назначение                     | Пример                 |
|--------|--------------------------------|------------------------|
| TODO   | задача на будущее              | `// TODO: исправить`   |
| FIXME  | известная ошибка               | `// FIXME: null check` |
| HACK   | временное решение / костыль    | `// HACK: workaround`  |

# 2. Архитектура и принцип работы

Расширение использует механизм **TextEditorDecorationType**
встроенный в VS Code API.


### Схема работы:

```text
Пользователь открывает файл
          │
          ▼
VS Code активирует расширение
          │
          ▼
activate()
  ├─ создаёт три стили подсветки
  ├─ регистрирует команду
  ├─ подписывается на события:
  │     • изменение документа
  │     • переключение вкладки
  └─ вызывает updateDecorations()
          │
          ▼
updateDecorations()
  ├─ читает текст файла
  ├─ находит TODO/FIXME/HACK по regex
  ├─ вычисляет диапазоны Range
  └─ применяет стили setDecorations()
